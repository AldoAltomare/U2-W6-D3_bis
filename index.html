SASS

Compiler per CSS, non viene letto dai browser

I partials sono quei file che iniziano col trattino basso "_"

I partials servono a scrivere codice riferito solo a quei determinati elementi (es. _header.scss, _footer.scss etc...)

Solitamente sono contenuti in una cartella "partials"

Poi si importano nel file scss in cima con: @import "nomecartella/nomefile"
